# BuDoBase

BuDoBase unterstützt die operative Betreuung von Kindern während eines Turnus.

## Erste Hilfe

**EH-Eintrag**:
Eine chronologische Dokumentation einer Erste-Hilfe-Maßnahme für genau ein Kind, bestehend aus einer verpflichtenden Beschreibung, erfassender Person, Erstellzeitpunkt und optionalen EH-Fotos.
_Avoid_: Notiz, Unfallprotokoll

**EH-Foto**:
Ein einem EH-Eintrag zugeordnetes Foto, das die Erste-Hilfe-Dokumentation ergänzt.
_Avoid_: Anhang, Notizfoto
